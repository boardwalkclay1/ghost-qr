    How to use GhostBoards QR Studio

    1. Paste the care page or how‑to URL into “Target URL”.
    2. (Optional) Add a video URL if you want a specific how‑to video referenced.
    3. Set the title and short instructions that will print under the QR.
    4. Choose the QR size (3×3 is default for board stickers) and color.
    5. Click “Generate QR” to update the preview.
    6. Click “Print 3×3 Label” to print a 3×3‑inch QR label, or download PNG/SVG.
    7. Save presets so you can reuse the same layout anytime.
